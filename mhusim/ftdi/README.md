* FDTI module used 
  [UMFT422EV-D](./doc/DS_UMFT4222EV.pdf).
  
* UMFT422EV-D jumpers settings
 
  We need to set FT4222H to CNFMODE0 {DCNF1, DCNF0} = 00
  configured by JP2 and JP3 ( See Figure 5-2 of [UMFT422EV-D data sheet ](./doc/DS_UMFT4222EV.pdf)  )
  
* Used pins
  - SCL  JP4 pin 4
  - SDA  JP4 pin 5
  - VBUS JP4 pin 1
  - GND  JP4 pin 3
